📋 guild-backend

Sistema de Gestão de Raides e Organizador de Atividades para Guilda

🎯 Descrição

Backend + Interface Web para administração de eventos,
escalas e participantes de uma guilda de jogos (focado em WoW WotLK).
Servidor em Node.js que fornece painel para líderes,
controle de presença e persistência de dados de raides.

🚀 Tecnologias Utilizadas

Node.js + JavaScript
HTML5 / CSS3
Express.js (servidor web)
Git / GitHub
Deploy pronto para plataformas como Render

✨ Funcionalidades

✅ Servidor local/remoto para acesso ao sistema
✅ Cadastro e gerenciamento de jogadores
✅ Criação e edição de raides e tamanhos de grupo
✅ Painel exclusivo para líderes de evento
✅ Layout responsivo adaptado a telas diferentes
✅ Persistência de dados das atividades

⚙️ Como Executar

bash
Run
# Clonar o repositório
git clone https://github.com/ILeekhan/guild-backend.git

# Entrar na pasta
cd guild-backend

# Instalar dependências
npm install

# Iniciar servidor
node server.js
Acesse em: http://localhost:3000
